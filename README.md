## 🔐 Безопасность
В этой версии добавлена базовая защита для продакшена:

- шифрование текстов сообщений и постов в базе данных (`enc:v1` через Fernet/AES);
- автоматическое шифрование старых plaintext-сообщений при старте;
- отдельный `MESSAGE_ENCRYPTION_KEY` в `.env`;
- настройка CORS через `CORS_ORIGINS`;
- базовый rate-limit для `/api` и `/api/auth`;
- security headers и HSTS в `ENVIRONMENT=production`.

Сгенерировать ключи для сервера:
```bash
python - <<'PY'
import secrets
from cryptography.fernet import Fernet
print('SECRET_KEY=' + secrets.token_urlsafe(64))
print('MESSAGE_ENCRYPTION_KEY=' + Fernet.generate_key().decode())
PY
```

Важно: `MESSAGE_ENCRYPTION_KEY` нельзя терять или менять без миграции — иначе старые сообщения нельзя будет расшифровать.

Это **не E2E-шифрование как в секретных чатах**: сервер всё ещё расшифровывает сообщения для выдачи пользователям. Но если утечёт только база данных, тексты сообщений будут защищены.

## ✅ Основные исправления
1. Каналы: фото, видео и стикеры отображаются без сжатого пузыря.
2. Комментарии: один общий чат на канал, комментарии как replies к постам.
3. Реакции на посты: меню реакций доступно и не уходит за экран.
4. Хронология каналов: новые посты идут вниз.
5. Голосовые: запись, предпрослушивание, правильная длительность, отправка кнопкой или повторным нажатием микрофона.
